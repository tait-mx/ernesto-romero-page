Set un conjunto conforme a igualdad


Hereda de: Object : Collection 


Un Set (Conjunto) es una colección de objetos de los cuales no hay dos iguales.

La mayoría de sus métodos son heredados de Collection.

El contenido de un Set no está ordenado. No puedes contar con el orden de los elementos en un Set.

Para un Set ordenado, ver OrderedIdentitySet.


Ver también: IdentitySet, List, Dictionary



Añadiendo y Quitando:


add(anObject)


Añade un objeto (a través del argumento anObject) al Set. No será añadido ningún objeto que sea igual a un objeto ya existente en el Set.


Set[1, 2, 3].add(4).postln;


Set[1, 2, 3].add(3).postln;


Set["abc", "def", "ghi"].add("jkl").postln;


Set["abc", "def", "ghi"].add("def").postln;


remove(anObject)


Quita un objeto del Set.


Set[1, 2, 3].remove(3).postln;



Iteración:


do(function)


Evalúa la función por cada elemento en el Set.

A la función se le envían dos argumentos, el elemento y un índice entero.


Set[1, 2, 3, 300].do({ arg item, i; item.postln });


keyAt(index)


devuelveel objeto en el índice interno. El índice no es determinista.



Operaciones específicas de Set:



sect(that) devuelve el conjunto intersección teórica de this y that

this & that


a = Set[1, 2, 3]; b = Set[2, 3, 4, 5];

sect(a, b);

a & b // sintaxis más corta


union(that) devuelve el conjunto unión teórica de this y that

this | that


a = Set[1, 2, 3]; b = Set[2, 3, 4, 5];

union(a, b);

a | b // sintaxis más corta


difference(that) devuelve el conjunto de todos los elementos de this, pero no de that

this - that


a = Set[1, 2, 3]; b = Set[2, 3, 4, 5];

difference(a, b);

a - b // sintaxis más corta


symmetricDifference(that) devuelve el conjunto de todos los elementos que no son elementos ni de this ni de that

this -- that


a = Set[1, 2, 3]; b = Set[2, 3, 4, 5];

symmetricDifference(a, b);

a -- b // sintaxis más corta


isSubsetOf(that) devuelve true (cierto) si todos los elementos de this también son elementos de that


a = Set[1, 2, 3, 4];

Set[1, 2].isSubsetOf(a); // cierto

Set[1, 5].isSubsetOf(a); // falso




// ejemplo:


a = Set[1, 2, 3, 4];

b = a.powerset; // conjunto de todas las partes

a.isSubsetOf(b); // falso: ningún conjunto es parte de sí mismo

b.asArray.reduce(\union) == a; // cierto: las partes no pueden contener elementos distintos a los originales

b.asArray.reduce(\difference).isEmpty; // cierto


// puedes usar Set para quitar de forma eficiente duplicaciones en un Array (Arreglo):


a = [1, 2, 3, 4, 3, 5, 5, 2, 2, 1];

a.as(Set); // convertir a Set

a.as(Set).as(Array); // y convertirlo de regreso

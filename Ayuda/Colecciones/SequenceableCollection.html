SequenceableCollection colección indexable por enteros


hereda de: Object : Collection


SequenceableCollection es una subclase de Collection cuyos elementos pueden ser indexados por un entero. Tiene muchas subclases útiles; Array y List están dentro de las mas comúnmente usadas.



Métodos de Clase


*series(size, start, step) // *serie(tamaño, inicio, paso)

Llena una SequenceableCollection con una serie aritmética.


Array.series(5, 10, 2);


*geom(size, start, grow) // *geom(tamaño, inicio, crecimiento)

Llena una SequenceableCollection con una serie geométrica.


Array.geom(5, 1, 3);


*fib(size, a, b) // *fib(tamaño, a, b)

Llena una SequenceableCollection con una serie de Fibonacci.

b es el valor de inicio (1 por defecto), a es el valor del paso inicial (0 por defecto)


Array.fib(5);

Array.fib(5, 2, 32); // inicia en 32 y el paso inicial de 2


*rand(size, minVal, maxVal) // *azar(tamaño, valMin, valMax)

Llena una SequenceableCollection  con valores al azar dentro del rango entre minVal y maxVal.


Array.rand(8, 1, 100);


*rand2(size, val) //  *azar2(tamaño, val)

Llena una SequenceableCollection  con valores al azar dentro del rango entre minVal y maxVal.


Array.rand2(8, 100);


*linrand(size, minVal, maxVal) // *azarlin(tamaño, valMin, valMax)

Llena una SequenceableCollection  con valores al azar dentro del rango entre minVal y maxVal con una 

distribución lineal.


Array.linrand(8, 1, 100);


*exprand(size, minVal, maxVal) // *azarexp(tamaño, valMin, valMax)

Llena una SequenceableCollection  con valores al azar dentro del rango entre minVal y maxVal con una 

distribución exponencial.


Array.exprand(8, 1, 100);


*interpolation(size, start, end) // *interpolación(tamaño, inicio, fin)

Llena una SequenceableCollection con los valores interpolados entre los valores de start y end.


Array.interpolation(5, 3.2, 20.5);



Métodos de Instancia


first //  primero

Devuelve el primer elemento de la collección, 


last //  último

Devuelve el último elemento de la collección, 


putFirst(item)   //  ponerPrimero(elemento)

putLast(item) //  ponerUltimo(elemento)

Pone el item en el primer o último índice de la collección. Notar que si la colección esta vacía (y por lo tanto no tiene espacios indexados) el item no será creado.


indexOf(item) // índice(elemento)

Devuelve el índice de un elemento (item) en la colección, o nil si no se encuentra.

indexOfEqual(item) // índiceDeIgual(elemento)

Devuelve el índice de algo que este en la colección que sea igual a item, o nil si no se encuentra.


indicesOfEqual(item) //  índicesDeIgual(elemento)

Devuelve un arreglo (array) de índices de cosas que esten en la colección y que sean iguales a item, o nil si no se encuentra.


y = [7, 8, 7, 6, 5, 6, 7, 6, 7, 8, 9]

y.indicesOfEqual(7);

y.indicesOfEqual(5);


indexOfGreaterThan(item) // índiceDeMayorQue(elemento)

Devuelve el primer índice que contenga un elemento mayor que item.


y = List[ 10, 5, 77, 55, 12, 123]; 

y.indexOfGreaterThan(70); 


find(sublist, offset) // ecnontrar(sublista, desplazamiento)

Si la sublist (sublista) existe en el receptor (en el orden especificado), en un desplazamiento (offset) mayor que o igual al offset inicial, entonces devuelve el índice de inicio.


y = [7, 8, 7, 6, 5, 6, 7, 6, 7, 8, 9]

y.find([7, 6, 5]);


findAll(arr, offset) //  encontrarTodos(arreglo, desplazamiento)

Similar a find() pero devuelve un arreglo con todos los índices en los que la secuencia  se encuentra


y = [7, 8, 7, 6, 5, 6, 7, 6, 7, 8, 9]

y.findAll([7, 6]);


indexIn(val) // índiceEn(valor)

devuelve el índice del valor mas cercano a val en la colección (la colección debe de estar ordenada)

[2, 3, 5, 6].indexIn(6.7)

indexInBetween(val) //  índiceEnMedio(val)

devuelve  un índice decimal interpolado linealmente para al valor (val)  (la colección debe de estar ordenada)

la operación inversa es blendAt

x = [2, 3, 5, 6].indexInBetween(5.2)

[2, 3, 5, 6].blendAt(x)



blendAt(floatIndex) //  mezclarEn(índicedecimal)

devuelve un valor interpolado linealmente entre los dos índices mas cercanos

la operación inversa es indexInBetween

x = [2, 5, 6].blendAt(0.4)



copyRange(start, end) //  copiarRango(inicio, fin)

Devuelve una nueva SequenceableCollection que es una copia de los espacios indexados del receptor desde start hasta end.

x.copyRange(a,b) también puede ser escrito como x[a..b]


(

var y, z;

z = [1, 2, 3, 4, 5];

y = z.copyRange(1, 3);

z.postln;

y.postln;

)


copyToEnd(start) // copiarHastaFinal(inicio)

Devuelve una nueva SequenceableCollection que es una copia de los espacios indexados del receptor desde start hasta el final de la colección. x.copyToEndtambién puede ser escrito como x[a..]


copyFromStart(end) //  copiarDesdeInicio(fin)

Devuelve una nueva SequenceableCollection que es una copia de los espacios indexados del receptor desde el principio de la colección hasta end. x.copyFromStart también se puede escribir como x[..a]


remove(item) //  remueve(elemento)

Remueve el item de la colección.


take(item) //  toma(elemento)

Remueve y devuelve  item de la colección. El último elemento de la colección se trasladará para ocupar el espacio vacante (y el tamaño de la colección decrece en uno). Ver tambien takeAt, definido en ArrayedCollection.


a = [11, 12, 13, 14, 15];

a.take(12);

a


keep(n) //  conserva(n)

Conserva los primeros n elementos del arreglo. Si n es negativo, conserva los últimos –n  elementos.


a = [1, 2, 3, 4, 5];

a.keep(3);

a.keep(-3);


drop(n) //  desecha(n)

Desecha los primeros n elementos del arreglo. Si n es negativo, desecha los últimos –n  elementos.


a = [1, 2, 3, 4, 5];

a.drop(3);

a.drop(-3);


join(joiner) //  junta(eslabón)

Devuelve un String formado al conectar todos los elementos del receptor, con joiner en medio.


["m", "ss", "ss", "pp", ""].join("i")


flat //  plano

Devuelve una colección en la que toda anidación ha sido aplanada (flattened)


[[1, 2, 3], [[4, 5], [[6]]]].flat;


flatten(numLevels) //  aplanar(númNiveles)

Devuelve una colección en la que cierto nivel de anidación, determinado por numLevel,  ha sido aplanado.


[[1, 2, 3], [[4, 5], [[6]]]].flatten(1).postcs;


[[1, 2, 3], [[4, 5], [[6]]]].flatten(2).postcs;


flop //  combina

IInvierte filas y columnas en una Collection de dos dimensiones (de adentro hacia afuera).

Ver tambíen: Function.


[[1, 2, 3], [4, 5, 6]].flop;

[[1, 2, 3], [4, 5, 6], [7, 8]].flop; // los arreglos mas pequeños se envuelven (se dan la vuelta y vuelven a empezar)

[].flop; // el resultado es siempre 2-d


lace(length) //  enlazar(longitud)

Devuelve una nueva Collection cuyos elementos son secuencias entrelazadas de los elementos de las subcolecciones del receptor, de hasta un tamaño determinado por lenght. El receptor no cambia.


(

x = [ [1, 2, 3], 6, List["botellota", 'laureles']];

y = x.lace(12);

x.postln;

y.postln;

)


resamp0(newSize) //  remuest0(nuevoTamaño)

Devuelve una Collection del tamaño deseado, con valores remuestreados del receptor, espaciados equidistantememente  y sin interpolacion


[1, 2, 3, 4].resamp0(13);

[1, 2, 3, 4].resamp0(2);


resamp1(newSize) //  remuest1(nuevoTamaño)

Devuelve una Collection del tamaño deseado, con valores remuestreados del receptor, espaciados equidistantemente  y con interpolacion


[1, 2, 3, 4].resamp1(12);

[1, 2, 3, 4].resamp1(3);


choose //  escoger

Escoge  al azar un elemento de la colección.


[1, 2, 3, 4].choose;


wchoose //  pescoger

Escoge al azar un elemento de la colección usando una lista de probabilidades o pesos.

Los pesos deben de sumar 1.0


[1, 2, 3, 4].wchoose([0.1, 0.2, 0.3, 0.4]);


sort(function) //  ordena(función)

Odena los elementos de la colección usando el argumnto de la función de comparación.

La función debe tomar dos elementos como argumentos y devolver ture (verdadero) si el primer 

argumento debe ser acomodado antes del segundo argumento.

S la función es nil,  la siguiente función por defecto es usada.


{ arg a, b; a < b }



[6, 2, 1, 7, 5].sort;


[6, 2, 1, 7, 5].sort({ arg a, b; a > b }); // orden en reversa



sortBy(key) // ordenaPor(clave)

Ordena los contenidos de la colección usando la clave key, la cual se asume está dentro de cada elemento del receptor.


(

a = [

Dictionary[\a->5, \b->1, \c->62], 

Dictionary[\a->2, \b->9, \c->65], 

Dictionary[\a->8, \b->5, \c->68], 

Dictionary[\a->1, \b->3, \c->61], 

Dictionary[\a->6, \b->7, \c->63], 

]

)

a.sortBy(\b);

a.sortBy(\c);



order(function) // ordenar(función)

Devuelve un arreglo de índices que acomodará la colección en orden. function es tratado del mismo modo que con el método sort.



[6, 2, 1, 7, 5].order;



swap(i, j) //  intercambiar(i, j)

Intercabiar dos elementos en la colección en los índices i y j.


pairsDo(function) //  paresHacer(función)

Llama la función para cada par de elementos subsecuentes en la SequenceableCollection.

La función pasa los dos elementos y un índice.


[1, 2, 3, 4, 5].pairsDo({ arg a, b; [a, b].postln; });


doAdjacentPairs(function) //  hacerParesAdyacentes (función)

Llama la función para cada par de elementos adyacentes en la SequenceableCollection.

La función pasa los dos elementos adyacentes y un índice.


[1, 2, 3, 4, 5].doAdjacentPairs({ arg a, b; [a, b].postln; });


separate(function) //  separar(función)

Separa la colección en subcolecciones llamando la función para cada par adyacente de elemntos.

Si la función responde true (verdadero), entonces se hace una separación entre los elementos.


 [1, 2, 3, 5, 6, 8, 10].separate({ arg a, b; (b - a) > 1 }).postcs;


clump(groupSize) //  grupo(TamañoDelGrupo)

Separa la colección en sub-colecciones separando cada groupSize elements.


[1, 2, 3, 4, 5, 6, 7, 8].clump(3).postcs;


clumps(groupSizeList) //  grupos(ListaDeTamañoDeGrupo)

Separa la colección en sub-colecciones separando los elementos en grupos cuyos tamaños estan dados por enteros en el groupSizeList.


[1, 2, 3, 4, 5, 6, 7, 8].clumps([1, 2]).postcs;


curdle(probability) //  grumos(probabilidad)

Separa la colección en sub-colecciones separando los elementos de acuerdo a una 

probabilidad dada.


[1, 2, 3, 4, 5, 6, 7, 8].curdle(0.3).postcs;


integrate //  integrar

Devuelve una colección con la suma incremental de todos los elementos


[3, 4, 1, 1].integrate;


differentiate //  diferenciar

Devuelve una colección con las diferencias por pares de todos os elementos.


[3, 4, 1, 1].differentiate;


reduce(operator) //  reducir(operador)

Aplica el método mencionado por operator al primero y segundo elemento de la colección - después aplica el método al resultado y al tercer elemento de la colección - después applica el método al resultado y al cuarto elemento de la colección - y asi continúa hasta el final del arreglo. operator puede ser Function o Symbol.



[3, 4, 5, 6].reduce('*'); // esto es lo mismo que [3, 4, 5, 6].product

[3, 4, 5, 6].reduce(\lcm); // mínimo común múltiplo de todo el conjunto de números

["d", "e", (0..9), "h"].reduce('++'); // concatenación

[3, 4, 5, 6].reduce({ |a, b| sin(a) * sin(b) }); // producto de senos



convertDigits(base) //  cinvierteDígitos(base)

Devuelve un entero resultante de la interpretación de los elemntos como dígitos de una base dada (10 por defecto).

Ver también asDigits en Integer para el método complementario.


[1, 0, 0, 0].convertDigits;

[1, 0, 0, 0].convertDigits(2);

[1, 0, 0, 0].convertDigits(3);


hammingDistance(that) // distanciaHamming(eso)

Devuelve una cuentaelementos de arreglos que no son iguales en posiciones idénticas. http://en.wikipedia.org/wiki/Hamming_distance


Las colecciones no son envueltas (wraped) - si un arreglo es mas corto que otro, la diferencia en tamaño debe ser incluida en la cuenta


[0, 0, 0, 1, 1, 1, 0, 1, 0, 0].hammingDistance([0, 0, 1, 1, 0, 0, 0, 0, 1, 1])

"SuperMan".hammingDistance("SuperCollider");

Mensajes Unarios:


Todos los mensajes siguientes mandan el mensaje performUnaryOp al receptor con el selector 

del mensaje unitario como argumento. 


neg, reciprocal, bitNot, abs, asFloat, asInt, ceil, floor, frac, sign, squared, cubed, sqrt

exp, midicps, cpsmidi, midiratio, ratiomidi, ampdb, dbamp, octcps, cpsoct, log, log2, 

log10, sin, cos, tan, asin, acos, atan, sinh, cosh, tanh, rand, rand2, linrand, bilinrand, 

sum3rand, distort, softclip, nyqring, coin, even, odd, isPositive, isNegative, 

isStrictlyPositive, real, imag, magnitude, magnitudeApx, phase, angle, rho, theta, 

asFloat, asInteger


performUnaryOp(aSelector) // realizaOperaciónBinaria(unSelector)

Crea una nueva colección con los resultados de aplicar el selector a todos los elementos del receptor.


[1, 2, 3, 4].neg;


[1, 2, 3, 4].reciprocal;


Mensajes Binarios:


Todos los mensajes siguientes mandan el mensaje performBinaryOp al receptor con el selector del mensaje binario y el segundo operando como argumentos.


+, -, *, /, div, %, **, min, max, <, <=, >, >=, &, |, bitXor, lcm, gcd, round, trunc, atan2, 

hypot, >>, +>>, fill, ring1, ring2, ring3, ring4, difsqr, sumsqr, sqrdif, absdif, amclip, 

scaleneg, clip2, excess, <!, rrand, exprand


performBinaryOp(aSelector, theOperand) // realizaOperaciónBinaria(unSelector, elOperando)

Crea una nueva colección de los resultados de aplicar el selector con el operando a todos los elementos en el receptor.

Si el operando es una colección entonces los elementos de esa colección se ponen en pares con los elementos del receptor.


([1, 2, 3, 4] * 10);


([1, 2, 3, 4] * [4, 5, 6, 7]);

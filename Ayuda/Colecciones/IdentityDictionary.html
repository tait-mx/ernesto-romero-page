IdentityDictionary colección asociativa que relaciona claves con valores



Hereda de: Object : Collection : Set : Dictionary


Un IdentityDictionary (Diccionario de Identidad) es una colección asociativa que relaciona claves con valores.

Las claves corresponden sólo si son objetos idénticos. (i.e. === devuelve true (cierto))

(En Dictionary, las claves correspondenden si son valoradas equivalentes. Esto hace a IdentityDictionary más rápido que Dictionary)


Ver también: Environment, Event (más documentación de los métodos en las superclases)


Frecuentemente, la subclase Event es usada como un IdentityDictionary, debido a que hay un acceso rápido sintáctico:


a = (); // devuelve un nuevo Event.

a.put(\foo, 2.718);

a.at(\foo);



Creación / Métodos de Clase


*new (n, proto, parent, know)

Las variables de instancia parent y proto permiten que IdentityDictionaries adicionales 

proporcionen valores por defecto. El orden de precedencia para determinar el valor de 

una clave es: el IdentityDictionary, su prototipo, su padre.


Cuando la variable de instancia know es true (cierto), el IdentityDictionary responde a 

mensajes desconocidos buscando el selector y evaluando el resultado con el diccionario 

como argumento.

Por ejemplo:


a = IdentityDictionary(know: true);

a.put(\foo, { | x | ("el argumento es:" + x).postln });

a.foo;

Métodos


IdentityDictionary reimplementa los siguientes métodos de Dictionary:

at (key)

put (key, value)

includesKey (key)

findKeyForValue (argValue)

scanFor (argKey)


putGet (key, newValue)

asigna la clave al nuevo valor (el argumento newValue), devuelve el valor anterior de la clave


Los siguientes tres métodos dan soporte para  Quant

nextTimeOnGrid (clock)

asQuant

timingOffset



Ejemplos


IdentityDictionary es frecuentemente usado para asignar nombres a instancias de una clase

particular. Por ejemplo, las clases "proxy" (Pdef, Pdefn, Tdef, Ndef, Fdef), SynthDefLib, y 

NodeWatcher, todas tienen variables de clase con el nombre all implementadas como 

IdentityDictionaries.

SuperCollider        Ayuda


Los archivos de ayuda son archivos HTML guardados en el directorio Help del folder de la aplicación SuperCollider.

Si eres un novato absoluto por favor remítete también el archivo README que vino al bajar (download) la aplicación.

Un navegador de ayuda puede ser abierto al ejecutar la siguiente línea de código:

Help.gui; //para correr solo una  línea de código pon el cursor en cualquier parte de la línea y ejecuta

Para ejecutar código, usa el atajo (shortcut) del teclado: Enter (no Return) (OS X), Ctrl+Enter (Windows), Ctrl+c Ctrl+c (Emacs). Para comandos con teclas, también ve Shortcuts.

Para explorar usando ligas (links) (que usualmente son azules y están subrayadas), simplemente haz click en ellas, o pide ayuda después de haberlas seleccionado, usa el menu Help, o teclea el atajo (shortcut) relevante: Cmd+d (OS X), F1 (Windows), Ctrl-c Ctrl-h (Emacs).

SuperCollider usa clases, cuyos nombres empiezan con mayúsculas todos. El archivo de ayuda de alguna clase puede ser pedido seleccionando el texto del nombre de una clase y después pidiendo la ayuda. También puedes navegar el directorio de clases:

Object.browse; //ejecuta esta línea de código

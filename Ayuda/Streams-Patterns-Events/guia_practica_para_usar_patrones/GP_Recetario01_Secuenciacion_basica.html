Recetario: Secuenciación básica


Reproduciendo una secuencia de notas predefinida


Las siguientes son tres diferentes maneras de tocar el famoso sujeto de una fuga.


El primero es con fuerza bruta, enlistando todos los grados de la escala mecánicamente en orden. El segundo y tercero reconocen el punto del pedal de La y usan operaciones de entrelazado para insertar las notas pedal entre las notas cambiantes.


El ejemplo demuestra el uso de las llaves de evento 'scale' y 'root' (escala y fundamental) para definir una tonalidad. Root = 2 es Re y la escala define el modo menor natural. La secuencia de grados ('degree') también usa alteraciones cromáticas. Restando 0.1 de un grado de escala entero vuelve la nota bemol; añadiendo 0.1 la sube medio tono. -0.9 es 0.1 más alto que -1, un grado de escala menor abajo de la tónica es una séptima bemol y medio tono arriba es la sensible: 


(

TempoClock.default.tempo = 84/60;


p = Pbind(

\scale, #[0, 2, 3, 5, 7, 8, 10],

\root, 2,

\degree, Pseq(#[rest, 4, 3, 4, 2, 4, 1, 4, 0, 4, -0.9, 4, 0, 4, 1, 4, 2, 4,

-3, 4, -1.9, 4, -0.9, 4, 0, 4, -0.9, 4, 0, 4, 1, 4, 2], 1),

\dur, 0.25

).play;

)


(

p = Pbind(

\scale, #[0, 2, 3, 5, 7, 8, 10],

\root, 2,

\degree, Place([#[rest, 3, 2, 1, 0, -0.9, 0, 1, 2, -3, -1.9, -0.9, 0, -0.9, 0, 1, 2],

(4 ! 16) ++ \rest], 17),

\dur, 0.25

).play;

)


(

p = Pbind(

\scale, #[0, 2, 3, 5, 7, 8, 10],

\root, 2,

\degree, Ppatlace([Pseq(#[rest, 3, 2, 1, 0, -0.9, 0, 1, 2, -3, -1.9, -0.9, 0, -0.9, 0, 1, 2], 1),

Pn(4, 16)], inf),

\dur, 0.25

).play;

)



Expansión "multicanal"


En una SynthDef, usar un arreglo como entrada a un UGen expande el UGen en un arreglo de UGens (ver MultiChannel). Algo similar ocurre con patrones. Normalmente un valor enviado a un nodo de Sinte debe ser un número sencillo, pero si en lugar es un arreglo, el patrón expande el evento para producir múltiples nodos de sinte en lugar de sólo uno. 


El patrón 'degree' aplica un conjunto de intervalos de acorde a una melodía que está siempre por encima. Es un patrón compuesto, Pseries(...) + Prand(...), donde Pseries retorna un número sencillo y Prand retorna un arreglo. Como con las operaciones regulares matemáticas, un número más un arreglo es un arreglo. Si el valor del Pseries actual es 7 y Prand retorna [0, -3, -5], el resultado es [7, 4, 2] y se escucha una triada de Do mayor en primera inversión. 


(

p = Pbind(

\degree, Pseries(7, Pwhite(1, 3, inf) * Prand(#[-1, 1], inf), inf).fold(0, 14)

+ Prand(#[[0, -2, -4], [0, -3, -5], [0, -2, -5], [0, -1, -4]], inf),

\dur, Pwrand(#[1, 0.5], #[0.8, 0.2], inf)

).play;

)


p.stop;



Usando SynthDefs a la medida (incluyendo SynthDefs sin altura definida)


Los patrones tienen características especiales que admiten varios estilos de organización de alturas, pero estas características son estrictamente opcionales. Aquí tocamos un SynthDef que no tiene ningún argumento de frecuencia.


Nótese el uso de memStore para preparar el SynthDef. Sin él, la mayoría de las entradas del SynthDef no serán reconocidas y el patrón será incapaz de mandarles valores. 


Es valioso notar que el patrón corre en pulsos, cuya duración real en segundos depende de el tempo del reloj. Sin embargo, el SynthDef siempre mide el tiempo en segundos. Este ejemplo mantiene las cosas simples al ajustar el reloj a un pulso por segundo. Si el tempo necesita ser algo diferente, la llave 'time' deberá ser dividida por el tempo: 


\time, Pkey(\delta) / Pfunc { thisThread.clock.tempo },



(

b = Buffer.read(s, "sounds/a11wlk01.wav");


SynthDef(\stretchedFragments, { |out, bufnum, start, time = 1, stretch = 1, amp = 1, attack = 0.01, decay = 0.05|

var sig = PlayBuf.ar(1, bufnum, rate: stretch.reciprocal, startPos: start, doneAction:2);

sig = PitchShift.ar(sig, pitchRatio: stretch)

* EnvGen.kr(Env.linen(attack, time, decay), doneAction: 2);

Out.ar(out, sig ! 2)

}).memStore; // nota: memStore! Sin esto, los argumentos no funcionarán!

)


(

TempoClock.default.tempo = 1;


p = Pbind(

\instrument, \stretchedFragments,

\bufnum, b,

\start, Pwhite(0, (b.numFrames * 0.7).asInteger, inf),

\delta, Pexprand(0.2, 1.5, inf),

\time, Pkey(\delta),

\stretch, Pexprand(1.0, 4.0, inf),

\amp, 0.5,

\attack, 0.1,

\decay, 0.2

).play;

)


p.stop;

b.free; // hay que ser ordenado, acordarse de limpiar el Buffer al final

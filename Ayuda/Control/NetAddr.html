NetAddr dirección de red


superclase: Object



*new(hostname, port) crea una nueva dirección de red.

Hostname es un string o cadena de caracteres, puede ser un número de ip (e.g. "192.168.34.56")

Port es un número de puerto, como 57110.

Nota: para mandar mensajes internamente, se usa el ip ¨127.0.0.1¨ que devuelve los mensajes a la misma computadora

*fromIP(ip, port) crea una nueva dirección de red usando un número de ip entero.

*langPort Obtiene el puerto por el que esta escuchando actualmente el sclang (puede cambiar después de una

recompilación)

*localAddr Obtiene  una NetAddr que corresponde con el localhost y el puerto por el que el sclang esta escuchando.

sendMsg(args...) envía un mensaje sin marca de tiempo (timestamp) a la dirección.

sendBundle(timestamp, args...) envía un paquete de datos con marca de tiempo.

sendRaw(rawArray) envía un mensaje en bruto, o sin procesar, sin marca de tiempo a la dirección.

connect(disconnectHandler) abre una conección TCP. disconnectHandler es llamado cuando  

se cierra la conexión (ya sea por el servidor o por el cliente)

disconnect cierra la conexión TCP

ip devuelve el número de ip (como cadena de caracteres o string)

ejemplo: 

n = NetAddr("localhost", 57110);

n.ip;

*disconnectAll cierra todas las conexiónes de TCP

*broadcastFlag checa la bandera de transmisión (broadcast flag)

*broadcastFlag_( flag ) Define la bandera de transmisión (sin importar si se puede mandar un mensaje broadcast o no)

*useDoubles_( flag )



// ejemplo



n = NetAddr("127.0.0.1", 57120); // 57120 es el puerto del sclang por defecto

r = OSCresponder(n, '/buenas/noticias', { arg time, resp, msg; [time, msg].postln }).add;


n.sendMsg("/buenas/noticias", "tu", "tu no");

n.sendMsg("/buenas/noticias", 1, 1.3, 77);



n.sendBundle(0.2, ["/buenas/noticias", 1, 1.3, 77]);


r.remove;

n.disconnect;


// notar que objetos NetAddr diferentes con el mismo puerto e ip son independientes


r = OSCresponder(nil, '/x', { "mensaje recibido".postln }).add;


n = NetAddr("127.0.0.1", 57120);

n.sendMsg("/x")



u = NetAddr("127.0.0.1", 57120);

u.sendMsg("/x");


n.disconnect


u.sendMsg("/x");


r.remove;

u.disconnect;

EnvGen generador de envolventes



Hereda de: Object : AbstractFunction : UGen



Reproduce envolventes con puntos de quiebre. Los envolventes son instancias de la clase Env. Para más información, ve el archivo de ayuda de Env. Los argumentos para levelScale, levelBias, y timeScale son solicitados cuando el EnvGen es disparado y permanecen constantes por la duración del envolvente.



*ar(envelope, gate, levelScale, levelBias, timeScale, doneAction)

*kr(envelope, gate, levelScale, levelBias, timeScale, doneAction)



envelope – una instancia de Env, o un Arreglo de Controles. (Ver Control y el ejemplo debajo para ver cómo usar esto.)

gate – Esto dispara el envolvente y lo mantiene abierto mientras sea mayor a 0. Si la longitud del envolvente es fija (p. ej. Env.linen, Env.perc) el argumento gate es usado como un gatillo simple. Si es un envolvente de sostenimiento, (p. ej. Env.adsr, Env.asr), el envolvente se mantiene abierto hasta que gate sea 0, momento en el cual es liberado.

levelScale – escala de los niveles de los puntos de quiebre.

levelBias – desface de los niveles de los puntos de quiebre.

timeScale – escala de la duración de los segmentos.

doneAction – un entero representando una acción a ser ejecutada cuando el env ha terminado su reproducción. Esto puede ser útil para liberar al synth que lo engloba, etc. Para más detalles, ver UGen-doneActions.



{ EnvGen.kr(Env.perc, 1.0, doneAction: 2) * SinOsc.ar(440, 0, 0.1) }.play;



// ejemplo



(

SynthDef(\env_help, { | out, gate = 0 | 

var z;

z = EnvGen.kr(Env.adsr, gate) * SinOsc.ar(440, 0, 0.1);

Out.ar(out, z)

}).send(s);

)

a = Synth(\env_help);

// encender

a.set(\gate, 1);

// apagar

a.set(\gate, 0);

// el valor de gate no importa, mientras sea mayor a 0

a.set(\gate, 2);

a.free;

Liberación forzada del EnvGen



Si el gate del EnvGen es -1 o menos, entonces el envolvente se cerrará inmediatamente. El tiempo para realizar el corte es la diferencia entre el valor y -1, con -1 siendo lo más rápido posible, -1.5 un corte en 0.5 segundos, etc. La forma del corte es lineal.



(

SynthDef(\stealMe, { arg gate = 1;

Out.ar(0, {BrownNoise.ar}.dup * EnvGen.kr(Env.asr, gate, doneAction:2))

}).send(s);

)

s.sendMsg(\s_new, \stealMe, 1001, 1, 0);

s.sendMsg(\n_set, 1001, \gate, -1.1); // corte en 0.1 segundos

Si el synthDef tiene un argumento llamado “gate”, el método de conveniencia de Node puede ser usado:

Node-release(releaseTime)

d = { arg gate=1; {BrownNoise.ar}.dup * EnvGen.kr(Env.asr, gate, doneAction:2) }.play;

d.release(3);
